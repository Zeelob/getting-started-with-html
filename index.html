Notes: The Basics of the Web and HTML

world wide web = consists of HTML (Hyper Text Markup Language).
- invented in the 1990's
- hyper links = connect HTML documents
- browser -> internet -> servers
- http = protocol

- elements form the html documents and have opening and closing tags
- html attributes (eg. links)
- html images = void tags = don't have closing tags, because no contents
- br = break, void tag + inline element
- p = paragraph, has content + block element (forms invisible block)
